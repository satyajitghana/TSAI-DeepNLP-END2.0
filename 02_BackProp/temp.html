Explanation

To solve/train a neuron network, we are using gradient descent, which is

Wnew=Wold−η∇EtotalW_{new}=W_{old}-\eta\nabla{E_{total}}Wnew​=Wold​−η∇Etotal​

where

Wnew=new weightsWold=old weightsη=learning rateEtotal=error totalW_{new} = \text{new weights} \\ W_{old}= \text{old weights} \\ \eta = \text{learning rate} \\ E_{total} = \text{error total}Wnew​=new weightsWold​=old weightsη=learning rateEtotal​=error total

Now we have weights W=[w1,w2,w3,w4,w5,w6,w7,w8]W = \lbrack w_1, w_2, w_3, w_4, w_5, w_6, w_7, w_8 \rbrackW=[w1​,w2​,w3​,w4​,w5​,w6​,w7​,w8​]

and

∇Etotal=∇Et=[∂Et∂w1,∂Et∂w2,∂Et∂w3,∂Et∂w4,∂Et∂w5,∂Et∂w6,∂Et∂w7,∂Et∂w8]\nabla E_{total} = \nabla E_t = \lbrack \frac{\partial E_t}{\partial w_1}, \frac{\partial E_t}{\partial w_2}, \frac{\partial E_t}{\partial w_3}, \frac{\partial E_t}{\partial w_4}, \frac{\partial E_t}{\partial w_5}, \frac{\partial E_t}{\partial w_6}, \frac{\partial E_t}{\partial w_7}, \frac{\partial E_t}{\partial w_8} \rbrack∇Etotal​=∇Et​=[∂w1​∂Et​​,∂w2​∂Et​​,∂w3​∂Et​​,∂w4​∂Et​​,∂w5​∂Et​​,∂w6​∂Et​​,∂w7​∂Et​​,∂w8​∂Et​​]

Now see carefully, the below equations, along with our network diagram above somewhere, very important!

h1=w1ii+w2i2h2=w3i1+w4i2h_1=w_1 i_i+w2 i_2 \\ h_2=w_3i_1+w4i_2 h1​=w1​ii​+w2i2​h2​=w3​i1​+w4i2​

ah1=σ(h1)ah2=σ(h2) a_{h1} = \sigma({h_1}) \\ a_{h2} = \sigma({h_2}) ah1​=σ(h1​)ah2​=σ(h2​)

o1=w5ah1+w6ah2o2=w7ah1+w8ah2 o_1 = w_5a_{h_1}+w_6a_{h_2} \\ o_2 = w_7a_{h_1}+w_8a_{h_2} o1​=w5​ah1​​+w6​ah2​​o2​=w7​ah1​​+w8​ah2​​

ao1=σ(o1)ao2=σ(o2) a_{o_1} = \sigma(o_1) \\ a_{o_2} = \sigma(o_2) ao1​​=σ(o1​)ao2​​=σ(o2​)

E1=12(t1−ao1)2.E2=12(t2−ao1)2 E_1=\frac{1}{2}(t_1-a_{o_1})^2 \\ .\\ E_2=\frac{1}{2}(t_2-a_{o_1})^2 E1​=21​(t1​−ao1​​)2.E2​=21​(t2​−ao1​​)2

Where [i1,i2] and [t1,t2][i_1, i_2] \text{ and }[t_1, t_2][i1​,i2​] and [t1​,t2​] are the inputs and target outputs respectively

Now we can start to compute the partial derivatives w.r.t to the weights, and also remember to use chain rule wherever applicable and necessary.

∂Et∂w5=∂E1∂w5=∂E1∂ao1∂ao1∂o1∂o1∂w5\frac{\partial E_t}{\partial w_5} = \frac{\partial E_1}{\partial w_5} = \frac{\partial E_1}{\partial a_{o_1}} \frac{\partial a_{o_1}}{\partial o_1} \frac{\partial o_1}{\partial w_5}∂w5​∂Et​​=∂w5​∂E1​​=∂ao1​​∂E1​​∂o1​∂ao1​​​∂w5​∂o1​​

We didn’t consider E2 above, because it does no contribution to w5

∂E1∂ao1=∂12(t1−ao1)2∂ao1=ao1−t1\frac{\partial E_1}{\partial a_{o_1}} = \frac{\partial \frac{1}{2}(t_1-a_{o_1})^2}{\partial a_{o_1}} = a_{o_1}-t_1∂ao1​​∂E1​​=∂ao1​​∂21​(t1​−ao1​​)2​=ao1​​−t1​

∂ao1∂o1=∂σ(o1)∂o1=σ(o1)(1−σ(o1))=(ao1)(1−ao1)\frac{\partial a_{o_1}}{\partial o_1}=\frac{\partial \sigma{(o_1)} }{\partial o_1} = \sigma(o_1)(1-\sigma(o_1)) = (a_{o_1})(1-a_{o_1})∂o1​∂ao1​​​=∂o1​∂σ(o1​)​=σ(o1​)(1−σ(o1​))=(ao1​​)(1−ao1​​)

and
∂o1∂w5=a−o1\frac{\partial o_1}{\partial w_5} = a-o_1∂w5​∂o1​​=a−o1​

similarly we can complete it for other weights in the last hidden layer

∂Et∂w5=(ao1−t1)(ao1)(1−ao1)ah1\frac{\partial E_t}{\partial w_5} = (a_{o_1}-t_1)(a_{o_1})(1-a_{o_1})a_{h_{1}}∂w5​∂Et​​=(ao1​​−t1​)(ao1​​)(1−ao1​​)ah1​​
∂Et∂w6=(ao1−t1)(ao1)(1−ao1)ah2\frac{\partial E_t}{\partial w_6} = (a_{o_1}-t_1)(a_{o_1})(1-a_{o_1})a_{h_{2}}∂w6​∂Et​​=(ao1​​−t1​)(ao1​​)(1−ao1​​)ah2​​
∂Et∂w7=(ao2−t2)(ao2)(1−ao2)ah1\frac{\partial E_t}{\partial w_7} = (a_{o_2}-t_2)(a_{o_2})(1-a_{o_2})a_{h_{1}}∂w7​∂Et​​=(ao2​​−t2​)(ao2​​)(1−ao2​​)ah1​​
∂Et∂w8=(ao2−t2)(ao2)(1−ao2)ah2\frac{\partial E_t}{\partial w_8} = (a_{o_2}-t_2)(a_{o_2})(1-a_{o_2})a_{h_{2}}∂w8​∂Et​​=(ao2​​−t2​)(ao2​​)(1−ao2​​)ah2​​

Now coming to the first hidden layer weights

∂Et∂w1=∂Et∂ao1∂ao1∂o1∂o1∂ah1∂ah1∂h1∂h1∂w1\frac{\partial E_t}{\partial w_1} = \frac{\partial E_t}{\partial a_{o_1}} \frac{\partial a_{o_1}}{\partial o_1 } \frac{\partial o_1}{\partial a_{h_1}} \frac{\partial a_{h_1}}{\partial h_1} \frac{\partial{h_1}}{\partial{w_1}}∂w1​∂Et​​=∂ao1​​∂Et​​∂o1​∂ao1​​​∂ah1​​∂o1​​∂h1​∂ah1​​​∂w1​∂h1​​

Seen this somewhere, right, we did this before

∂Et∂ao1∂ao1∂o1∂o1∂ah1=∂(E1+E2)∂ah1=∂E1∂ah1∂E2∂ah1.=(ao1−t1)(ao1)(1−ao1)w5+(ao2−t2)(ao2)(1−ao2)w7\frac{\partial E_t}{\partial a_{o_1}} \frac{\partial a_{o_1}}{\partial o_1 } \frac{\partial o_1}{\partial a_{h_1}} = \frac{\partial (E_1+E_2)}{\partial a_{h_1}}= \frac{\partial E_1}{\partial a_{h_1}} \frac{\partial E_2}{\partial a_{h_1}} \\ . \\ = (a_{o_1}-t_1)(a_{o_1})(1-a_{o_1})w_5 \\ + (a_{o_2}-t_2)(a_{o_2})(1-a_{o_2})w_7∂ao1​​∂Et​​∂o1​∂ao1​​​∂ah1​​∂o1​​=∂ah1​​∂(E1​+E2​)​=∂ah1​​∂E1​​∂ah1​​∂E2​​.=(ao1​​−t1​)(ao1​​)(1−ao1​​)w5​+(ao2​​−t2​)(ao2​​)(1−ao2​​)w7​

Now we can compute all the gradient of the first layer

∂Et∂w1=∂Et∂ah1(a−h1)(1−ah1)i1\frac{\partial E_t}{\partial w_1} = \frac{\partial E_t}{\partial a_{h_1}} (a-h_1)(1-a_{h_1})i_1∂w1​∂Et​​=∂ah1​​∂Et​​(a−h1​)(1−ah1​​)i1​
∂Et∂w2=∂Et∂ah1(a−h1)(1−ah1)i2\frac{\partial E_t}{\partial w_2} = \frac{\partial E_t}{\partial a_{h_1}} (a-h_1)(1-a_{h_1})i_2∂w2​∂Et​​=∂ah1​​∂Et​​(a−h1​)(1−ah1​​)i2​
∂Et∂w3=∂Et∂ah2(a−h2)(1−ah2)i1\frac{\partial E_t}{\partial w_3} = \frac{\partial E_t}{\partial a_{h_2}} (a-h_2)(1-a_{h_2})i_1∂w3​∂Et​​=∂ah2​​∂Et​​(a−h2​)(1−ah2​​)i1​
∂Et∂w4=∂Et∂ah2(a−h2)(1−ah2)i2\frac{\partial E_t}{\partial w_4} = \frac{\partial E_t}{\partial a_{h_2}} (a-h_2)(1-a_{h_2})i_2∂w4​∂Et​​=∂ah2​​∂Et​​(a−h2​)(1−ah2​​)i2​
